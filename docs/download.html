log4j version 1.0.1 (2001-01-10 13:45:15 UTC)

Download the complete package including full source code, class files and documentation in TGZ format or in ZIP format. You may elect to browse log4j documentation online.

Version 1.0.1 is the 21th public release of log4j.

Thanks to Henri Gomez, Linux users can now fetch log4j in RPM format at http://rpmized.free.fr or alternatively at ftp://ftp.falsehope.com/home/gomez/log4j/.

We also maintain a list of earlier versions of log4j for download, intended for the curious paleontologist.
